# Projet E-commerce

Projet e-commerce réalisé en programmation full stack.

## Technologies utilisées
- Frontend : Angular
- Backend : Node.js avec Express
- Base de données : db.json (JSON)

## Fonctionnalités
- Page Welcome (Sign In / Register)
- Authentification utilisateur
- Catalogue de produits avec filtres
- Détails d’un produit
- Ajout au panier avec gestion du stock
- Affichage du panier (quantité, prix, total)
- Possibilité de vider le panier
- Déconnexion (Log Out)
